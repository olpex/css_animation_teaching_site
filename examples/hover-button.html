Кнопка з ефектом наведення

Створення інтерактивної кнопки з ефектом підняття при наведенні курсора

<button class="hover-button">Наведіть на мене</button>
.hover-button {
  padding: 12px 24px;
  background: #3498db;
  color: white;
  border: none;
  border-radius: 4px;
  transition: transform 0.3s ease,
              box-shadow 0.3s ease;
  font-size: 16px;
  cursor: pointer;
}

.hover-button:hover {
  transform: translateY(-5px);
  box-shadow: 0 5px 15px rgba(0,0,0,0.2);
}

Як це працює

Ця кнопка використовує CSS переходи для створення плавного ефекту підняття при наведенні курсора. Ось ключові компоненти:

  • transition: Визначає, які властивості будуть анімовані (transform і box-shadow), тривалість анімації (0.3 секунди) та функцію часу (ease).
  • transform: translateY(-5px): При наведенні курсора кнопка піднімається на 5 пікселів вгору.
  • box-shadow: Додає тінь під кнопкою, щоб підсилити ефект підняття.

Цей ефект створює відчуття, що кнопка "спливає" при наведенні курсора, що робить інтерфейс більш інтерактивним і привабливим для користувача.

Переваги використання CSS переходів:

  • Плавна анімація без використання JavaScript
  • Висока продуктивність завдяки апаратному прискоренню
  • Простий і зрозумілий код
  • Покращення користувацького досвіду

Варіації ефекту

.scale-button {
  padding: 12px 24px;
  background: #9b59b6;
  color: white;
  border: none;
  border-radius: 4px;
  transition: transform 0.3s ease;
}

.scale-button:hover {
  transform: scale(1.1);
}
.color-button {
  padding: 12px 24px;
  background: #2ecc71;
  color: white;
  border: none;
  border-radius: 4px;
  transition: background-color 0.3s ease,
              color 0.3s ease;
}

.color-button:hover {
  background-color: white;
  color: #2ecc71;
}
.shadow-button {
  padding: 12px 24px;
  background: #e74c3c;
  color: white;
  border: none;
  border-radius: 4px;
  transition: box-shadow 0.3s ease;
}

.shadow-button:hover {
  box-shadow: 0 0 0 5px rgba(231, 76, 60, 0.4);
}
