Підкреслення при наведенні

Створення анімованого підкреслення для посилань при наведенні курсора

Наведіть на мене 
<a href="#" class="underline-link">Наведіть на мене</a>
.underline-link {
  position: relative;
  color: #78909c;
  text-decoration: none;
  font-size: 1.5rem;
  font-weight: 500;
}

.underline-link::after {
  content: '';
  position: absolute;
  bottom: -2px;
  left: 0;
  width: 100%;
  height: 2px;
  background-color: #b0bec5;
  transform: scaleX(0);
  transform-origin: right;
  transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.underline-link:hover::after {
  transform: scaleX(1);
  transform-origin: left;
}

Як це працює

Цей ефект створює анімоване підкреслення, яке з'являється зліва направо при наведенні курсора на посилання. Ось ключові компоненти:

  • position: relative: Встановлює позиціонування для батьківського елемента, щоб псевдоелемент ::after міг бути позиціонований відносно нього.
  • ::after: Псевдоелемент, який використовується для створення лінії підкреслення.
  • transform: scaleX(0): Початково лінія має нульову ширину (невидима).
  • transform-origin: right: Встановлює точку початку трансформації з правого краю.
  • transition: Визначає, як анімувати зміну властивості transform.
  • transform: scaleX(1): При наведенні лінія розтягується до повної ширини.
  • transform-origin: left: При наведенні змінює точку початку трансформації на лівий край, створюючи ефект "заповнення" зліва направо.

Цей ефект створює елегантне анімоване підкреслення, яке привертає увагу користувача та покращує інтерактивність інтерфейсу.

Переваги використання цього підходу:

  • Плавна анімація без використання JavaScript
  • Можливість налаштування товщини, кольору та швидкості анімації
  • Покращення користувацького досвіду
  • Сумісність з усіма сучасними браузерами
