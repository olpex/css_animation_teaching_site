Інтерактивні анімації

Створюйте анімації, що реагують на дії користувача

touch_app

Що таке інтерактивні анімації?

Інтерактивні анімації реагують на дії користувача, такі як наведення курсору, клік, фокус або прокрутка. Вони покращують взаємодію користувача з інтерфейсом, роблячи його більш інтуїтивним та приємним.

Анімації при наведенні та фокусі

Найпростіший спосіб додати інтерактивність - це використовувати псевдокласи :hover та :focus для запуску анімацій при наведенні курсору або фокусуванні на елементі.

CSS 
.hover-button {
  padding: 12px 24px;
  background-color: var(--primary-color);
  color: white;
  border: none;
  border-radius: 4px;
  font-size: 1rem;
  cursor: pointer;
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.hover-button:hover {
  transform: translateY(-5px);
  box-shadow: 0 5px 15px rgba(98, 0, 238, 0.3);
}

.hover-button:focus {
  outline: none;
  box-shadow: 0 0 0 3px rgba(98, 0, 238, 0.5);
}

Демонстрація ефекту наведення

Наведіть курсор на кнопку, щоб побачити анімацію.

accessibility

Доступність

Завжди забезпечуйте, щоб інтерактивні елементи мали стани :focus, які відповідають станам :hover, для користувачів, які використовують клавіатуру для навігації.

Анімації при кліку

Для створення анімацій, що запускаються при кліку, можна використовувати JavaScript для додавання та видалення класів, які містять анімації.

HTML 
<div class="click-box" id="clickBox"></div>
CSS 
.click-box {
  width: 100px;
  height: 100px;
  background-color: var(--primary-color);
  border-radius: 8px;
  margin: 20px 0;
  cursor: pointer;
  transition: background-color 0.3s ease;
}

.click-box.active {
  background-color: var(--secondary-color);
  animation: pulse 1s infinite;
}

@keyframes pulse {
  0% {
    transform: scale(1);
  }
  50% {
    transform: scale(1.1);
  }
  100% {
    transform: scale(1);
  }
}
JavaScript 
document.addEventListener('DOMContentLoaded', function() {
  const clickBox = document.getElementById('clickBox');
  
  clickBox.addEventListener('click', function() {
    this.classList.toggle('active');
  });
});

Демонстрація анімації при кліку

Клікніть на квадрат, щоб запустити або зупинити анімацію.

Інтеграція з JavaScript

JavaScript дозволяє створювати більш складні інтерактивні анімації, керуючи CSS властивостями або використовуючи Web Animation API.

JavaScript (Web Animation API) 
const element = document.getElementById('animatedElement');

const keyframes = [
  { transform: 'translateX(0)', opacity: 1 },
  { transform: 'translateX(50px)', opacity: 0.5, offset: 0.7 },
  { transform: 'translateX(100px)', opacity: 0 }
];

const options = {
  duration: 1000,
  easing: 'ease-in-out',
  fill: 'forwards'
};

// Запуск анімації
const animation = element.animate(keyframes, options);

// Керування анімацією
animation.pause();  // Пауза
animation.play();   // Відтворення
animation.reverse(); // Реверс
animation.finish(); // Завершення

Анімаційні події

CSS анімації генерують події, які можна відслідковувати за допомогою JavaScript для створення складних послідовностей анімацій.

JavaScript 
const element = document.getElementById('animatedElement');

element.addEventListener('animationstart', function(e) {
  console.log('Анімація почалася: ' + e.animationName);
});

element.addEventListener('animationend', function(e) {
  console.log('Анімація завершилася: ' + e.animationName);
  
  // Запуск наступної анімації після завершення поточної
  this.classList.remove('animation1');
  this.classList.add('animation2');
});

element.addEventListener('animationiteration', function(e) {
  console.log('Ітерація анімації: ' + e.animationName);
});
lightbulb

Порада

Використовуйте подію animationend для створення послідовностей анімацій, де кожна наступна анімація запускається після завершення попередньої.
