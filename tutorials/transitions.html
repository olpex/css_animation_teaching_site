Вступ до CSS переходів

CSS переходи дозволяють плавно змінювати значення властивостей протягом заданого періоду часу. Вони є найпростішим способом додати анімацію до вашого веб-сайту без використання JavaScript або складних анімаційних бібліотек.

Відеоурок: В цьому відео ви дізнаєтесь про основи CSS переходів, як вони працюють та як їх використовувати для створення плавних анімацій на вашому веб-сайті.

Переходи особливо корисні для:

  • Анімації при наведенні курсора
  • Плавної зміни кольорів
  • Зміни розміру елементів
  • Створення простих інтерактивних ефектів

Основи CSS переходів

Для створення переходу потрібно вказати:

  1. Властивість, яку ви хочете анімувати
  2. Тривалість переходу
  3. Функцію часу (необов'язково)
  4. Затримку перед початком (необов'язково)

Базовий синтаксис

.element {
  transition-property: property;
  transition-duration: duration;
  transition-timing-function: timing-function;
  transition-delay: delay;
}

/* Або скорочений запис */
.element {
  transition: property duration timing-function delay;
}

Демонстрація

Наведіть курсор на квадрат

.transition-demo-box {
  width: 100px;
  height: 100px;
  background-color: #3498db;
  transition: background-color 0.5s ease;
}

.transition-demo-box:hover {
  background-color: #e74c3c;
}

Властивості, які можна анімувати

Не всі CSS властивості можна анімувати. Ось список найбільш часто використовуваних властивостей, які підтримують переходи:

Кольори

  • color
  • background-color
  • border-color
  • outline-color

Розміри

  • width
  • height
  • padding
  • margin

Позиціонування

  • top
  • right
  • bottom
  • left

Трансформації

  • transform
  • transform-origin

Примітка: Властивості, які не мають проміжних значень (наприклад, display), не можуть бути анімовані за допомогою переходів.

Функції часу

Функції часу визначають, як швидко змінюється значення властивості протягом переходу. CSS надає кілька вбудованих функцій часу:

ease (за замовчуванням)

linear

ease-in

ease-out

ease-in-out

Приклад використання функцій часу

.element {
  transition: transform 1s ease-in-out;
}

.element:hover {
  transform: translateX(100px);
}

Ви також можете створювати власні функції часу за допомогою cubic-bezier():

.element {
  transition: transform 1s cubic-bezier(0.68, -0.55, 0.27, 1.55);
}

Анімація кількох властивостей

Ви можете анімувати кілька властивостей одночасно, вказавши їх через кому або використовуючи ключове слово all:

Анімація кількох властивостей

.button {
  background-color: #3498db;
  color: white;
  padding: 10px 20px;
  border-radius: 4px;
  transition: background-color 0.3s ease,
              transform 0.3s ease,
              box-shadow 0.3s ease;
}

.button:hover {
  background-color: #2980b9;
  transform: translateY(-3px);
  box-shadow: 0 5px 15px rgba(0,0,0,0.2);
}

Демонстрація

Порада: Використовуйте all з обережністю, оскільки це може вплинути на продуктивність, особливо на складних сторінках.

Приклади використання

Кнопка з ефектом наведення

.example-button {
  padding: 12px 24px;
  background-color: #3498db;
  color: white;
  border: none;
  border-radius: 4px;
  transition: background-color 0.3s ease;
}

.example-button:hover {
  background-color: #2980b9;
}

Картка з ефектом підняття

Картка з переходом

Наведіть курсор, щоб побачити ефект

.example-card-hover {
  padding: 20px;
  background-color: white;
  border-radius: 8px;
  box-shadow: 0 2px 5px rgba(0,0,0,0.1);
  transition: transform 0.3s ease,
              box-shadow 0.3s ease;
}

.example-card-hover:hover {
  transform: translateY(-10px);
  box-shadow: 0 15px 30px rgba(0,0,0,0.2);
}
