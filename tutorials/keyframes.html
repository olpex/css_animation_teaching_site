Анімації з ключовими кадрами

Опануйте складні анімації з @keyframes для необмежених можливостей

movie

Що таке анімації з ключовими кадрами?

CSS анімації з ключовими кадрами дозволяють створювати складні послідовності анімацій, визначаючи різні стани елемента в різні моменти часу. На відміну від переходів, анімації з ключовими кадрами можуть автоматично повторюватися, змінювати напрямок та мати більш складну поведінку.

Основи @keyframes

Правило @keyframes визначає послідовність анімації, вказуючи, як елемент повинен виглядати в різні моменти часу. Ось базовий синтаксис:

CSS 
@keyframes назва-анімації {
  0% {
    /* Стилі для початку анімації */
  }
  50% {
    /* Стилі для середини анімації */
  }
  100% {
    /* Стилі для кінця анімації */
  }
}

Після визначення ключових кадрів, ви можете застосувати анімацію до елемента за допомогою властивості animation:

CSS 
.елемент {
  animation-name: назва-анімації;
  animation-duration: 3s;
  animation-timing-function: ease-in-out;
  animation-delay: 0s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
  animation-fill-mode: forwards;
  animation-play-state: running;
}

Або використовуючи скорочений запис:

CSS 
.елемент {
  animation: назва-анімації 3s ease-in-out 0s infinite alternate forwards running;
}

Демонстрація анімації з ключовими кадрами

Рух по квадрату

Ця анімація переміщує елемент по квадратній траєкторії, змінюючи колір на кожному кроці.

CSS 
.keyframes-demo-box {
  width: 100px;
  height: 100px;
  background-color: var(--primary-color);
  border-radius: 8px;
  animation: moveAround 4s infinite;
}

@keyframes moveAround {
  0% {
    transform: translateX(0) translateY(0);
    background-color: var(--primary-color);
  }
  25% {
    transform: translateX(200px) translateY(0);
    background-color: var(--secondary-color);
  }
  50% {
    transform: translateX(200px) translateY(100px);
    background-color: var(--accent-color);
  }
  75% {
    transform: translateX(0) translateY(100px);
    background-color: var(--success-color);
  }
  100% {
    transform: translateX(0) translateY(0);
    background-color: var(--primary-color);
  }
}

Порада

Ви можете використовувати ключові слова from і to замість 0% і 100% для простих анімацій з двома станами.

Властивості анімації

CSS надає кілька властивостей для контролю анімацій:

  • animation-name: Ім'я анімації, визначене в @keyframes
  • animation-duration: Тривалість одного циклу анімації
  • animation-timing-function: Функція часу (ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier)
  • animation-delay: Затримка перед початком анімації
  • animation-iteration-count: Кількість повторень (число або infinite)
  • animation-direction: Напрямок анімації (normal, reverse, alternate, alternate-reverse)
  • animation-fill-mode: Стан елемента до і після анімації (none, forwards, backwards, both)
  • animation-play-state: Стан відтворення (running, paused)
