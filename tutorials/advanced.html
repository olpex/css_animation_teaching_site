Просунуті техніки анімації

У цьому уроці ми розглянемо складніші техніки CSS анімації, які дозволять вам створювати більш вражаючі та інтерактивні ефекти для ваших веб-проектів. Ви дізнаєтесь про анімацію SVG, паралакс ефекти, анімацію при прокрутці та оптимізацію продуктивності.

Анімація SVG

SVG (Scalable Vector Graphics) - це формат векторної графіки, який ідеально підходить для анімації, оскільки він масштабується без втрати якості та має доступ до окремих елементів для анімації.

HTML 
<svg width="200" height="200" viewBox="0 0 200 200">
  <circle class="animated-circle" cx="100" cy="100" r="50" fill="none" stroke="#6200ee" stroke-width="4"/>
</svg>
CSS 
.animated-circle {
  stroke-dasharray: 314;
  stroke-dashoffset: 314;
  animation: draw 2s linear forwards;
}

@keyframes draw {
  to {
    stroke-dashoffset: 0;
  }
}
lightbulb

Порада

Для анімації SVG часто використовуються властивості stroke-dasharray і stroke-dashoffset, які дозволяють створювати ефект малювання ліній.

Паралакс ефекти

Паралакс ефект створює ілюзію глибини, коли різні елементи сторінки рухаються з різною швидкістю при прокрутці.

Демонстрація паралакс ефекту

Наведіть курсор на зображення, щоб побачити ефект паралаксу.

JavaScript 
document.addEventListener('DOMContentLoaded', function() {
  const parallaxContainer = document.getElementById('parallaxDemo');
  const layers = parallaxContainer.querySelectorAll('.parallax-layer');
  
  parallaxContainer.addEventListener('mousemove', function(e) {
    const x = e.clientX / parallaxContainer.offsetWidth;
    const y = e.clientY / parallaxContainer.offsetHeight;
    
    layers.forEach((layer, index) => {
      const speed = (layers.length - index) * 20;
      const moveX = (x - 0.5) * speed;
      const moveY = (y - 0.5) * speed;
      
      layer.style.transform = `translate(${moveX}px, ${moveY}px)`;
    });
  });
});

Анімація при прокрутці

Анімація при прокрутці дозволяє запускати анімації, коли елементи з'являються у видимій області екрану.

CSS 
.fade-in {
  opacity: 0;
  transform: translateY(20px);
  transition: opacity 0.6s ease, transform 0.6s ease;
}

.fade-in.visible {
  opacity: 1;
  transform: translateY(0);
}
JavaScript 
document.addEventListener('DOMContentLoaded', function() {
  const fadeElements = document.querySelectorAll('.fade-in');
  
  function checkFade() {
    fadeElements.forEach(element => {
      const elementTop = element.getBoundingClientRect().top;
      const elementVisible = 150;
      
      if (elementTop < window.innerHeight - elementVisible) {
        element.classList.add('visible');
      }
    });
  }
  
  window.addEventListener('scroll', checkFade);
  checkFade(); // Перевірка при завантаженні сторінки
});

Оптимізація продуктивності

Для забезпечення плавних анімацій важливо оптимізувати їх продуктивність:

  • Використовуйте властивості transform і opacity - вони не викликають перекомпонування сторінки і анімуються найбільш плавно.
  • Уникайте анімації властивостей, що впливають на компонування - таких як width, height, margin, padding.
  • Використовуйте will-change - для підказки браузеру про елементи, які будуть анімуватися.
  • Обмежуйте кількість анімованих елементів - особливо на мобільних пристроях.
CSS 
.optimized-animation {
  will-change: transform, opacity;
  transform: translateZ(0); /* Активує апаратне прискорення */
}
warning

Застереження

Використовуйте will-change економно, оскільки надмірне використання може призвести до збільшення споживання пам'яті.
